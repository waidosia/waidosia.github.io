MongoDB

发表于 2021-05-16 分类于 数据库 阅读次数： Valine：
本文字数： 4.3k 阅读时长 ≈ 4 分钟

MongoDB 是一个基于分布式文件存储的数据库。由 C++ 语言编写。旨在为 WEB 应用提供可扩展的高性能数据存储解决方案。

MongoDB 是一个介于关系数据库和非关系数据库之间的产品，是非关系数据库当中功能最丰富，最像关系数据库的。

查询

主键值:数据库会自动添加主键，是一个12字节的bson类型字符串(主机名、进程id、时间戳等等)

时间戳:数据库存储的日期会自动转化成格林尼治时间

表达式

MongoDB的表达式必须是JSON格式
| 表达式 | 意义 | 表达式 | 意义 |
| :—: | :—: | :—: | :—: |
| $lt | 小于 | $gt | 大于 |
| $lte | 小于等于 | $gte | 大于等于 |
| $in | 包括 | $nin | 不包括 |
| $ne | 不等于 | $all | 全部 |
| $not | 全部 | $or | 或关系 |
| $exists | 含有关系 |

正则表达式

MongoDB支持正则表达式查找内容

分页查找数据

可以使用skip()函数设置起始量,limit()函数设置偏移量

数据排序

使用sort()函数排序，1代表升序，-1代表降序

排除重复

使用distinct()函数来替换不重复的记录

修改记录

update()和updateMany()函数实现对数据的修改

删除记录

remove()函数删除记录

索引机制

因为数据库存放大量的数据，所以为了加快数据检索速度，需要设置索引
索引字段按照升序排列，属性值为1;降序排列属性值为-1
创建索引会阻塞数据库，影响其他操作，所以background参数代表在空闲的时间创建索引

唯一性索引只能创建在每个记录都含有的公共字段上，非公共字段不能创建唯一性索引

创建索引的原则

  • 数据量恒大的集合必须创建索引
  • 集合的数据读取超过写入
  • 经常被当做查询条件的字段
